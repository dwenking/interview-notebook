## 文件系统

### 软链接和硬链接的区别？

|                    | 软链接                                                                  | 硬链接                                                                                     |
| ------------------ | ----------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| 是否支持跨文件系统 | 是                                                                      | 否                                                                                         |
| 索引节点           | 软链接原文件和链接文件拥有不同的inode号，<br />表明它们是两个不同的文件 | 硬链接原文件和链接文件共用同一个inode号，<br />表明它们是同一个文件                        |
| 主要作用           | 通常用于创建快捷方式，以方便用户访问文件或目录                          | 硬链接能够允许一个文件拥有多个有效路径名，<br />可用于创建备份，多个文件可以指向同一份数据 |
